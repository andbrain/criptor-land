# criptor-land
Criptador/Desencriptador Land usando pilhas

## Created by:
- @andbrain

## Installation:
g++ -I. *.cpp -o criptor

## How to use:
- To encrypt and decrypt, you have to run like:
	1. Encrypt:
    > ./criptor ola mundo
    > ndo muola
	2. Decrypt:
    > ./criptor ndo muola
    > ola mundo
